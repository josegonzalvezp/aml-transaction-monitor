# AML Transaction Monitor

## Overview

AML Transaction Monitor is a Python project that simulates a simplified Anti-Money Laundering (AML) transaction monitoring system.

The application analyses financial transactions, evaluates them against a configurable set of rules and generates alerts when the calculated risk score exceeds a predefined threshold.

The main objective of the project is to practise object-oriented programming, software architecture and Python best practices while developing a realistic financial application.

---

## Project Goals

- Design a clean object-oriented architecture.
- Implement an extensible rule engine.
- Apply software engineering principles such as SOLID.
- Write unit tests using pytest.
- Keep a professional Git history.
- Continuously improve the project by adding new AML features.
